# Fullstack Skill
## Purpose
Bridging frontend and backend technologies seamlessly.

## Scope
End-to-end features, monolithic repos, API mapping.

## Priority
High

## Rules
- Share types across frontend and backend boundaries.
- Co-locate logic only when modularity is maintained.
- Ensure end-to-end typing via tRPC or standard fetch wrappers.
- Deploy frontend and backend together if coupled.
