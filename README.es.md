# AI Engineering Workspace Kit

🇬🇧 [English](README.md) | 🇪🇸 [Español](README.es.md)

Un repositorio de inicio robusto y reutilizable, diseñado para inicializar nuevos proyectos con reglas de ingeniería IA, habilidades preferidas, flujos de trabajo necesarios y configuraciones esenciales. Esta configuración actúa como un Kit de Desarrollo para entornos asistidos por IA como Antigravity, Windsurf y sistemas similares.

## Propósito
El objetivo principal es proporcionar un punto de partida único y coherente para cualquier nuevo proyecto, cerrando la brecha entre los archivos típicos del proyecto y las instrucciones para la IA. Todas las reglas internas están estructuradas en un formato Markdown conciso y eficiente en tokens.

## Documentación
Para facilitar su uso, la documentación se ha separado estrictamente en guías modulares:

- [1. Primeros Pasos](docs/es/01_getting_started.md) — Instalación, inicialización y scripts de automatización.
- [2. Resumen de Arquitectura](docs/es/02_architecture.md) — Explicación de los módulos principales, reglas y configuraciones.
- [3. Gestión de Habilidades](docs/es/03_skills_management.md) — Cómo crear, registrar y validar habilidades (skills) de bajo consumo de tokens.
- [4. Proceso de Lanzamiento](docs/es/release-process.md) — Guías para el CI/CD y lanzamientos a producción.

---

> **Nota**: Este repositorio está preparado para la inicialización y desarrollo local y estructurado con modularidad, clara separación de responsabilidades y redundancia mínima. Mantén siempre las instrucciones internas en los archivos `.md` en **Inglés** según las directivas del repositorio.
