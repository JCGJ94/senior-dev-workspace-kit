# Agent Rules
## Purpose
Control how AI subagents operate and coordinate.

## Scope
Agent orchestration and task delegation.

## Priority
High

## Rules
- Use specialists (e.g., @codebaseInvestigator, @bugFixer) for specific tasks.
- Do not ask for permission before invoking necessary subagents.
- Ensure state continuity between subagent handoffs.
- Validate subagent output before merging or presenting.
