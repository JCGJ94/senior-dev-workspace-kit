# Specs Workspace

This directory stores Spec-Driven Development artifacts for non-trivial work.

Each change should use:

```text
specs/<change-id>/
  01-intake.md
  02-explore.md
  03-proposal.md
  04-spec.md
  05-design.md
  06-tasks.md
  07-implementation-log.md
  08-verification.md
  09-archive.md
  summary.md
```

The active change must remain concise and auditable.
